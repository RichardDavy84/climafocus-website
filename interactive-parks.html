Explore Climate Futures for a National Park in Northern Norway

Interactive data visualization tool for Climate Futures in Reisa National Park

Reisa National Park Explorer

This tool provides an overview of potential climate futures for Reisa National Park. It is oriented around the conservation priorities of the park managers e.g. Forest Fires, River summer access, and the user can select between these. Each priority has it's own set of climate drivers with defined safe, neutral, and dangerous ranges which are summarised at a glance using a traffic light system. You can adjust the time window and emissions scenario to see how the climate drivers of each priority may change over time.
